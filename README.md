# streamsrv

### Lib

Install `raylib` via this [https://github.com/raysan5/raylib#build-and-installation](https://github.com/raysan5/raylib#build-and-installation).

### Compile and run

```bash
make

./bin/server [server_port]

./bin/client [server_ip] [server_port] [rtp_port] [video_file]
```
